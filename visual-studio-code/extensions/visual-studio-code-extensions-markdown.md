# Visual Studio Code Markdown

- `⇧⌘V`: Open preview in another tab
- `⌘K V`: Open preview in a split
- Latex math is supported wrapped in `$` or `$$`
- To force reload of the Markdown Preview (e.g., if an image has change on disk and the preview is showing the old image, use `Show All Commands > Developer: Reload Webviews`
