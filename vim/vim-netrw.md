# `vim` `netrw`

- `:e`: Refresh the file list
- `cd`: Changes local directory to current directory (this does an `:lcd` not `:cd`)
- `d`: Create directory
- `D`: Delete file
- `R`: Rename (or move) file
- `%`: Create a new file at the current directory

With `vim-vinegar`, `gh` to show hidden files.
