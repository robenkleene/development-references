# BBEdit

- `⇧⌘U`: Show command palette
- The `Document Info Panel` can be shown by clicking the document icon in the upper right of the toolbar, it contains metadata about the file and the file can be renamed from it

## Bindings

- `⇧⌥⌫`: Forward delete word

## Clippings

Use `F5` for autocomplete to expand clippings.

## Columns

- To select multiple columns in a `csv` file, select a couple of columns, e.g., in a single row. Then choose `Edit > Columns > Select Columns`.
