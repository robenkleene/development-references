# BBEdit

- `⇧⌘U`: Show command palette

## Bindings

- `⇧⌥⌫`: Forward delete word

## Clippings

Use `F5` for autocomplete to expand clippings.

## Columns

- To select multiple columns in a `csv` file, select a couple of columns, e.g., in a single row. Then choose `Edit > Columns > Select Columns`.
